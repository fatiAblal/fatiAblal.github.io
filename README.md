# Hidden Figures (Il Diritto di Contare) - Static Website

## Descrizione

Progetto web molto semplice realizzato durante le prime fasi di apprendimento di HTML e CSS.

L'obiettivo era familiarizzare con la struttura di una pagina web, la gestione dei contenuti multimediali e la creazione di uno stile grafico di base attraverso CSS.

Il sito è dedicato al film "Hidden Figures" e contiene informazioni sul film, il cast e i principali riconoscimenti ottenuti.

## Tecnologie utilizzate

- HTML5
- CSS3

## Funzionalità

- Sito web statico multipagina
- Navigazione tramite collegamenti interni
- Inserimento di immagini e video
- Layout e formattazione tramite CSS
